# TRULY DARKMODE FOR VSCODE

### SETTINGS:

- Window zoom level
- Default tab size
- Color customizations

### HOW TO:

Add settings in **Preferences: settings (JSON)** file on VSCODE. 

### TESTED ON:

- VSCODE Windows 10
- WSL: Ubuntu-18.04
- WSL: Ubuntu-20.04

### NOTE:

Feel free to add, remove or edit settings to your liking in your machine.
